Philadelphia Food Delivery Database

For this project I worked with a team to create a Philadelphia food delivery service database. For this project my team and I had to create ERR diagrams to plan out all of the various people that would use the app and the database tables that would need to be created. Some of these tables included restraunts, menus, delivery drivers, customers, and many more. We then used PL/SQL to create all of these tables and created data to import into they system to show that we had a fully opertaional food delivery service database.

Bookstore Inventory System

For this project I had the responsibility of creating an inventory system for a Bookstore. To do this I used C# to create a checkout system where a customer could add a book to their cart and once it was added it would remove the book from inventory. If the customer took it out of their cart or 10 minutes has passed the book would be added back into the inventory. Once checkout was complete it would remove the book from the MySQL datbase and calculate the total which the customer had owed. It also allowed the customer to see all of the books that were in stock or out of stock in the database on a different form.
